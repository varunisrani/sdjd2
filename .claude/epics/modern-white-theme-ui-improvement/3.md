# Task 3: SearchBar & Navigation UX

## Description

Enhance SearchBar with mobile-friendly sizing and improve navigation patterns. This task focuses on refactoring the SearchBar component for optimal mobile experience and improving overall navigation UX with accessible focus states and responsive layouts.

## Acceptance Criteria

- Mobile: Full-width search with large touch target (44px height)
- Desktop: Maintain 320px width
- Focus state: 3px outline with high contrast
- Keyboard navigation fully supported
- Search results dropdown responsive

## Technical Details

### SearchBar Component Updates

- **Update `components/SearchBar.tsx`**
  - Refactor for responsive sizing (mobile-first)
  - Apply Tailwind responsive modifiers (base mobile, `md:` desktop)
  - Implement accessible focus states with WCAG 2.1 AA compliance
  - Add clear visual feedback for all interaction states

### Mobile Layout (320-640px)

- **SearchBar Container**
  - Width: 100% (full width)
  - Height: 44px (minimum touch target)
  - Padding: 12px (horizontal), 8px (vertical)
  - Margin: 12px (bottom spacing to content)

- **Input Field**
  - Font size: 16px (prevents iOS zoom on focus)
  - Height: 44px
  - Padding: 12px 16px
  - Border radius: 8px
  - Border: 1px solid (light gray, #E5E5E5)

- **Focus State**
  - Outline: 3px solid (primary color, #000000)
  - Outline offset: 2px
  - Smooth transition: 150ms ease-out

### Desktop Layout (641px+)

- **SearchBar Container**
  - Width: 320px (fixed width, no full-width)
  - Height: auto (let content drive height)
  - Positioning: Top navigation area

- **Input Field**
  - Font size: 14px (smaller than mobile for compact desktop layout)
  - Height: 40px
  - Padding: 10px 14px
  - Border radius: 6px

### Keyboard Navigation

- **Tab Navigation**
  - SearchBar receives focus via Tab key
  - Clear visual focus indicator (3px outline)
  - No focus trap (focus moves to next element naturally)

- **Keyboard Shortcuts** (optional enhancements)
  - `Cmd/Ctrl + K`: Focus SearchBar (if global shortcut implemented)
  - `Escape`: Clear search, close dropdown

- **Arrow Keys in Dropdown**
  - Up/Down arrows: Navigate search results
  - Enter: Select result
  - Escape: Close dropdown

### Dropdown Responsiveness

- **Mobile Dropdown**
  - Full-width below SearchBar (100% minus padding)
  - Max-height: 60vh (allows scrolling within dropdown)
  - Smooth slide-down animation (150ms, translate)

- **Desktop Dropdown**
  - Width: Match SearchBar (320px)
  - Max-height: 400px
  - Positioned below SearchBar with shadow

### Focus State Implementation

```css
.search-input:focus {
  outline: 3px solid;
  outline-color: var(--color-primary, #000000);
  outline-offset: 2px;
  transition: outline 150ms ease-out;
}
```
