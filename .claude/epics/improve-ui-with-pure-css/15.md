# Task: Migrate MusicPlayer to CSS Modules

## Description
Migrate the MusicPlayer component to use CSS Modules instead of Tailwind classes. Add smooth animations for play/pause button, enhance progress bar transitions, and style the volume slider with custom CSS.

## Acceptance Criteria
- [ ] Create `components/MusicPlayer.module.css` with all MusicPlayer styles
- [ ] Replace all Tailwind classes in MusicPlayer.tsx with CSS Module classes
- [ ] Play/Pause button has scale animation on click (1.0 â†’ 1.05)
- [ ] Progress bar has smooth transition on seek
- [ ] Volume slider styled with custom CSS (no default browser styles)
- [ ] Control buttons have hover states with scale + color change
- [ ] Sticky positioning at bottom works perfectly
- [ ] Mobile responsive: player adapts to small screens
- [ ] No visual regression from current design

## Technical Details
**Files to create:**
- `components/MusicPlayer.module.css`

**Files to modify:**
- `components/MusicPlayer.tsx` (replace all Tailwind classes)

**Implementation approach:**
1. Create MusicPlayer.module.css with class names:
   - `.player` - main container (fixed bottom)
   - `.playerContent` - inner content wrapper
   - `.trackInfo` - song info section
   - `.artwork` - album artwork
   - `.controls` - control buttons container
   - `.playButton` - play/pause button
   - `.controlButton` - skip/shuffle/repeat buttons
   - `.progressBar` - progress bar container
   - `.progressFill` - filled portion
   - `.volumeSlider` - volume control

2. Replace Tailwind classes with CSS Module classes

3. Implement animations:
   ```css
   .playButton:active {
     transform: scale(1.05);
     transition: transform 100ms ease-out;
   }

   .controlButton:hover {
     transform: scale(1.1);
     color: var(--primary);
   }
   ```

4. Style custom range input (volume slider):
   ```css
   .volumeSlider::-webkit-slider-thumb {
     /* Custom thumb styling */
   }
   ```

5. Add mobile responsive styles (<640px)

**Code locations affected:**
- `components/MusicPlayer.tsx` (entire component)
