# Task: Migrate main page layout to CSS Modules

## Description
Migrate the main page layout (app/page.tsx) and root layout (app/layout.tsx) to use CSS Modules instead of Tailwind classes. Implement CSS Grid for the main layout structure.

## Acceptance Criteria
- [ ] Create `app/page.module.css` for main page layout styles
- [ ] Replace all Tailwind classes in page.tsx with CSS Module classes
- [ ] Update layout.tsx body className (remove Tailwind utilities)
- [ ] Implement CSS Grid for sidebar + main content layout
- [ ] Responsive: layout stacks on mobile, side-by-side on desktop
- [ ] Header section styled with proper spacing and alignment
- [ ] Content area has proper padding and scrolling behavior
- [ ] Activity cards styled consistently
- [ ] No visual regression from current design

## Technical Details
**Files to create:**
- `app/page.module.css`

**Files to modify:**
- `app/page.tsx` (main layout structure, lines 108-231)
- `app/layout.tsx` (body className, line 32)

**Implementation approach:**
1. Create page.module.css with class names:
   - `.appContainer` - main app wrapper
   - `.mainContent` - content area
   - `.header` - header section
   - `.contentArea` - scrollable content
   - `.section` - content sections
   - `.activityCard` - activity card styling
   - `.grid` - grid layout for cards

2. Replace Tailwind in page.tsx:
   - `className="min-h-screen flex"` → `className={styles.appContainer}`
   - `className="flex-1 flex flex-col"` → `className={styles.mainContent}`
   - etc.

3. Implement CSS Grid layout:
   ```css
   .appContainer {
     display: grid;
     grid-template-columns: 256px 1fr;
     min-height: 100vh;
   }

   @media (max-width: 640px) {
     .appContainer {
       grid-template-columns: 1fr;
     }
   }
   ```

4. Update layout.tsx body:
   - Remove: `className={`${geistSans.variable} ${geistMono.variable} antialiased bg-white text-slate-900`}`
   - Add: `className={`${geistSans.variable} ${geistMono.variable}`}`
   - Move bg and text color to globals.css body rule

**Code locations affected:**
- `app/page.tsx` lines 108-231 (layout structure)
- `app/layout.tsx` line 32 (body tag)
