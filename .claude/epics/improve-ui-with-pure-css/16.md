# Task: Migrate TrackList to CSS Modules

## Description
Migrate the TrackList component to use CSS Modules instead of Tailwind classes. Add smooth row hover transitions, enhance active track highlighting, and ensure responsive column behavior.

## Acceptance Criteria
- [ ] Create `components/TrackList.module.css` with all TrackList styles
- [ ] Replace all Tailwind classes in TrackList.tsx with CSS Module classes
- [ ] Row hover has smooth background color transition (150ms)
- [ ] Active track highlighted with primary color background
- [ ] Play button fades in smoothly on row hover
- [ ] All action buttons are 44x44px touch targets
- [ ] Responsive: album column hidden on mobile (<640px)
- [ ] Table layout uses CSS Grid or Flexbox properly
- [ ] No visual regression from current design

## Technical Details
**Files to create:**
- `components/TrackList.module.css`

**Files to modify:**
- `components/TrackList.tsx` (replace all Tailwind classes)

**Implementation approach:**
1. Create TrackList.module.css with class names:
   - `.trackList` - main container
   - `.header` - table header
   - `.headerActions` - action buttons (play all, shuffle, etc.)
   - `.table` - track table container
   - `.trackRow` - individual track row
   - `.trackRowActive` - active/playing track
   - `.trackRow:hover` - hover state
   - `.playButtonCell` - play button column
   - `.trackInfo` - track title/artist
   - `.albumColumn` - album column (hidden on mobile)
   - `.durationColumn` - duration column

2. Replace Tailwind classes with CSS Module classes

3. Implement smooth transitions:
   ```css
   .trackRow {
     transition: background-color 150ms ease-out;
   }

   .trackRow:hover {
     background-color: var(--track-hover);
   }

   .trackRowActive {
     background-color: var(--primary-light);
   }
   ```

4. Responsive design:
   ```css
   @media (max-width: 640px) {
     .albumColumn {
       display: none;
     }
   }
   ```

5. Ensure action buttons are 44x44px minimum

**Code locations affected:**
- `components/TrackList.tsx` (entire component)
